## What You Need to Do Now

### Step 1: Install Dependencies (Required)
Open PowerShell in VS Code and run:

```powershell
npm install
```

‚è±Ô∏è **This will take 2-5 minutes**. This installs all required packages:
- React Native & Expo

- React Navigation
- Formik, Yup, Axios, AsyncStorage
- And many more...

### Step 2: Get API Key (Required)
1. Go to: **https://api-ninjas.com/**
2. Click "Sign Up" (it's free!)
3. Verify your email
4. Go to "API" section
5. Copy your API key

### Step 3: Configure API Key (Required)
1. Open: `src/services/apiService.ts`
2. Find line 3: `const API_NINJAS_KEY = 'YOUR_API_KEY_HERE';`
3. Replace `YOUR_API_KEY_HERE` with your actual API key
4. Save the file

### Step 4: Run the App
```powershell
npm start
```

Then:
- Press `a` for Android emulator
- Press `i` for iOS simulator (Mac only)
- Or scan QR code with Expo Go app on your phone

### Step 5: Test the App
1. **Register** a new account with any email
2. **Browse** exercises on Home screen
3. **Tap** an exercise to see details
4. **Add** to favourites
5. **Check** Profile to see your info
